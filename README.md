# 🎮 Miniguia de Game Design com IA (NotebookLM)

## 📖 Contexto

Este projeto foi desenvolvido como parte de um desafio prático da DIO com o objetivo de explorar o uso da Inteligência Artificial como ferramenta de aprendizado ativo.

O tema escolhido foi **Game Design**, com foco em entender como os jogos conseguem prender a atenção dos jogadores e gerar engajamento.

## 🎯 Objetivos

- Entender os principais conceitos de Game Design
- Aprender como funcionam mecânicas de engajamento
- Explorar o uso de IA (NotebookLM) para estudo
- Criar um material de revisão reutilizável

## 📘 Miniguia de Estudo

### 🧩 Principais Conceitos

- Game Loop: ciclo principal de ações do jogador
- Progressão: evolução dentro do jogo
- Recompensa: sistema que motiva o jogador
- Engajamento: o quanto o jogador continua jogando

## 📖 Glossário

- Game Loop: sequência repetitiva de ações no jogo
- XP (Experiência): pontos usados para progressão
- Level Design: criação de fases e ambientes
- Feedback: resposta do jogo às ações do jogador

## 💡 Prompts Reutilizáveis

- "Explique [conceito] de forma simples"
- "Dê exemplos práticos de [conceito] em jogos"
- "Compare dois sistemas de [mecânica]"
- "Resuma esse conteúdo em tópicos"
